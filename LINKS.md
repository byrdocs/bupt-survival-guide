# 机器人查询链接

该文件用于给 BYR Docs 机器人提供可查询的网址链接。文件不会部署到网站。链接描述可以简要总结页面的用途，包含什么类型的内容，但不要存放页面的具体内容，因为该内容可能随时间更新。

## 注意事项

fetch-mcp 默认情况抽取“可读正文”，适合普通新闻/文章页面; 但对**图片为主、表格为主、或自定义布局**的页面, Defuddle 会把主要内容当噪声丢掉。遇到这种情况，调用时加 `raw: true` 参数。下方链接标注 `raw=true` 的链接需要使用该参数获取，否则使用默认模式获取。

当添加一个新链接时，需要测试两种模式并记录在本文档中。

## 链接

### 北京邮电大学 - 办公电话

https://www.bupt.edu.cn/ggfw/bgdh.htm

北邮学校各部门,学院的办公电话。

### 北京邮电大学 - 校历

https://www.bupt.edu.cn/ggfw/xl.htm  (raw=true)

当用户提到”校历”时获取并发送。校历本身是两张 JPG 图片嵌在页面里：第一张为秋季学期校历，第二张为春季学期校历。发送时优先根据当前时间选择一张图片发送。

### 北京邮电大学 - 院系机构

https://www.bupt.edu.cn/yxjg1.htm  (raw=true)

北京邮电大学院系机构一览，列出各学院、行政部门等教学单位和职能部门的官方网站链接。
